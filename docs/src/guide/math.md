# 数学公式渲染指南

## 简介
当你在百度教育做题页面看到类似 `$\frac{1}{8}$` 这样的数学公式代码时，不用担心，只需按照以下步骤操作即可正确显示数学公式。

## 操作步骤

1. 在百度教育做题页面，找到包含数学公式代码的输入框
2. 右键点击输入框，在弹出菜单中选择：
   - "百度教育" → "渲染数学公式"

![数学公式渲染选项](../img/WechatIMG33.jpg)

3. 等待系统渲染（渲染时间取决于公式数量）

## 注意事项

- 数学公式必须用美元符号（$）包裹才能正确渲染
  - 正确示例：`$\frac{1}{8}$`
  - 错误示例：`\frac{1}{8}`
- 如果公式长时间未渲染，请刷新页面重试
- 页面中公式较多时，渲染可能需要更长时间，请耐心等待
